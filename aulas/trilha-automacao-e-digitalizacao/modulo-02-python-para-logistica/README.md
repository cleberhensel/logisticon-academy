# Módulo — Python para Logística

**Objetivo em uma linha (catálogo):** automatização de relatórios e rotinas — **ambiente** (*venv*, *requirements*), **segredos** fora do código, ***pandas*** em exports logísticos, **agendamento** e consumo **GET** REST com tratamento de erros — sem substituir a trilha Dados (Power BI avançado).

## Resultado do módulo

Esboçar **um** fluxo (descrição + pseudocódigo) que consolide **dois** ficheiros num **KPI** ou relatório semanal.

## Erros comuns do módulo

- Correr *script* em **produção** sem IT/ciber.  
- Chave API **commitada** no repositório.  
- *Script* «do analista» sem **versão** nem *owner*.
