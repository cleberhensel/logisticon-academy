# Módulo 2 — Excel Avançado para Logística

**Objetivo em uma linha:** dashboards, Power Query e modelagem tabular reprodutível para o dia a dia do CD e do planejamento.

---

## Objetivos de aprendizagem (mensuráveis)

1. Montar um modelo mini com **tabelas** relacionadas (fato + duas dimensões).
2. Construir um fluxo **Power Query** com *merge*/*append* e tipos corretos (datas, IDs).
3. Publicar um painel com **slicers** e cartões de KPI com **documentação de refresh**.

## Erros comuns

- «Congelar» valores copiados em vez de manter consultas ligadas.
- `PROCV` em bases que crescem sem chave estável.
- Misturar **linha** e **pedido** no mesmo denominador de serviço.

---

## Atividade de consolidação

Dashboard com **lista de suposições** e passos de *refresh* (origem → transformação → relatório).
