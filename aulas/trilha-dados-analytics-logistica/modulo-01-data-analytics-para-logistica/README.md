# Módulo 1 — Data Analytics para Logística

**Objetivo em uma linha:** dados para decisão logística — antes de abrir qualquer ferramenta, **pergunta**, **granularidade** e **qualidade**.

---

## Objetivos de aprendizagem (mensuráveis)

1. Formular uma pergunta de negócio **testável** e traduzi-la em colunas necessárias e *grain* do *dataset*.
2. Mapear fontes típicas (ERP, WMS, TMS, planilhas) e o **dono** de cada extração.
3. Identificar viés de **ruptura**, **promoção** e duplicidade entre bases.
4. Produzir um esboço de narrativa visual (gráfico justificado) para OTIF ou lead time.

## Erros comuns

- Começar pelo gráfico «bonito» sem definir denominador e janela temporal.
- Misturar *grain* (ex.: pedido e linha na mesma tabela sem regra).
- Tratar vendas como demanda real sem tratar **stockout**.

---

## Atividade de consolidação

**One-pager:** pergunta + *grain* + riscos de dados + um gráfico com legenda de decisão.
