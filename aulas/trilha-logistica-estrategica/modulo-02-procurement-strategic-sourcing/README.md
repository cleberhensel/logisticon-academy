# Módulo — Procurement e Strategic Sourcing

**Objetivo em uma linha (catálogo):** compras estratégicas — **estratégia por categoria**, **make-or-buy**, **TCO**, processo **RFP → award**, leilões como ferramenta e **risco/ESG** como restrição de *sourcing* (sem consultoria fiscal ou jurídica).

## Resultado do módulo

Definir estratégia plausível para **duas** categorias distintas (ex.: commodity padronizada *versus* item de engenharia) com indicadores de TCO e mitigação de risco nomeada.

## Erros comuns do módulo

- **Preço unitário** como única métrica de sucesso.  
- RFP **genérico** que não amarra escopo, SLA e critérios de award.  
- ESG como **slide de marketing** sem critério auditável no fornecedor.
