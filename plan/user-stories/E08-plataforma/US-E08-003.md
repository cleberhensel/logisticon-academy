# US-E08-003 — Export e exclusão de dados (LGPD mínima)

**Épico:** E08 · **Spec:** [SPEC-08](../../specs/SPEC-08-plataforma-notificacoes-lgpd.md) · **Task:** [TSK-DEV-049](../../tasks/TSK-DEV-049.md) · **Prioridade:** P1

**Como** titular dos dados  
**Quero** exportar meus dados e solicitar exclusão/anonimização  
**Para** exercer direitos previstos em lei, respeitando retenção fiscal.

## Critérios de aceite

- **Given** usuário autenticado **When** solicita export **Then** recebe pacote (ex.: JSON) com dados pessoais tratados pela plataforma, conforme política documentada.
- **Given** pedido de exclusão **When** processado **Then** PII é anonimizada ou removida onde permitido e registros financeiros obrigatórios permanecem conforme retenção; ação auditável no backoffice.
